# Let's Fly

This is a aviation management system for the [Let's Fly](https://letsfly.com) project. It is a web application that allows users to book their flights, manage their bookings, and manage their account.

### Project Setup

1. Clone the repository
2. Go to [client](./client/) folder and run `yarn install`, to install the dependencies in frontend.
3. Run `yarn dev` to start the application in development mode.
4. Go to [http://localhost:3000](http://localhost:3000) to view the application.
5. Now go to [server](./server/) folder and run `yarn install`, to install the dependencies in backend.
6. Run `yarn dev` to start the server in development mode.
7. Go to [http://localhost:5000](http://localhost:5000) to view the server.
8. Work with the API's in [http://localhost:5000/api](http://localhost:5000/api).
